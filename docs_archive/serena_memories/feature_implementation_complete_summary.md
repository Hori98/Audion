### 2. Auto-Pick機能  
**実装場所**: `audion-app/app/(tabs)/feed.tsx:232-277, 494-505`

**AI選択ロジック**:
- カテゴリー分散: 各カテゴリーから1-2記事を選択
- 日付優先: 新しい記事を優先的に選択
- コンテンツ品質: サマリー長による品質判定
- 最適な長さ: 最大5記事でポッドキャスト最適化

**ユーザーフロー**:
1. "Auto-Pick (X件から自動選択)" ボタン
2. AI選択実行 → 自動的に選択モード開始
3. 選択確認 → FABで音声作成
